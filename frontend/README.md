# GoalFlow — Enterprise Goal Setting & Tracking Portal

This is the frontend repository for GoalFlow. Please refer to the root `README.md` in `e:\AtomQuest\README.md` for the comprehensive project documentation, architecture, features, and setup instructions.

## Quick Start

1. Ensure Node.js (v20.18.0) is installed.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Access the application at `http://localhost:5174` (or `5173`).
